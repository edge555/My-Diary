# MyDiary

A simple web app for you dairy writing

## Built With

* [VS Code](https://code.visualstudio.com)

## Language Used
* **Front-End**
  * HTML
  * Bootstrap
  * Javascript
* **Back-End**
  * Node.js
  * Express.js
  * MongoDB

## Features

1. Notes : You can add, edit, delete notes
2. Login System : Only you will be able to access you account
